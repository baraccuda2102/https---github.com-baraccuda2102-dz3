Селекторы

У нас есть как минимум 5 селекторов

  • за названием елемента

    • Это выражения, которые говорят браузеру, к какому элементу HTML нужно применить те или иные свойства CSS

  • за ID

    • #id Ідентифікатор визначає унікальне ім'я елемента, яке використовується для зміни його стилю і звернення до нього через скрипти.

  • за классом

    • .class Класи застосовують, коли необхідно визначити стиль для одного або декількох елементів веб-сторінки. Елементи при цьому можуть бути різними, але мати частково або повністю схожі елементи оформлення. У коді HTML ім'я класу визначається через атрибут

  • за нескольками классами

    • .class1.class2 До будь-якого елемента одночасно можна додати кілька класів, перераховуючи їх в атрибуті class через пробіл. У цьому разі до елемента застосовується стиль, описаний у правилах для кожного класу. Оскільки під час додавання кількох класів вони можуть містити однакові стильові властивості, але з різними значеннями, то береться значення в класу, який описано в коді нижче.
